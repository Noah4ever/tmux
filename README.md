## Quickstart
1. Install all [dependencies](##Dependencies)
2. Open Tmux
3. Install all TPM plugins. Press `CRTL + Space` and then `SHIFT + I`

## Dependencies
- [TPM](https://github.com/tmux-plugins/tpm#installation)
- [Tmux](https://github.com/tmux/tmux/wiki/Installing)
